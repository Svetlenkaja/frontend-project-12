## HexletChat

[demo version](https://frontend-project-12-ex6p.onrender.com/)

### Setup

- clone the repository 
``` 
$ git clone https://github.com/Svetlenkaja/frontend-project-12.git
```

- install the package and all its dependencies
```sh
$ make install
```

- run on localhost
```
make start
```
